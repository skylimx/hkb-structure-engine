# **Glossary**

| **Term** | **Definition** |
| --- | --- |
| SCF | Structure Classification Framework. A framework for defining stable industrial structural dimensions. |
| IPD | Industrial Product Definition. A system for determining whether structural combinations constitute valid industrial product existence. |
| MAP | Marketplace Mapping. A mapping system between industrial structures and channel structures. |
| VIS | Visual Identity Structure. A visual cognition system for industrial structures. |
| Structure World | The structural abstraction world formed by industrial product structures and dimensions. |
| Industrial Existence | A structurally valid industrial product existence recognized within the industrial world. |
| SCF Node | The structural node at which SCF begins defining dimensions. |
| Structural Dimension | A stable industrial characteristic used for structural classification. |
| Structural Combination | A combination generated from multiple SCF dimensions. |
| Structural Validity | Whether a structural combination can form real industrial existence. |
| Channel World | The representation world formed by websites, marketplaces, listings, and navigation systems. |
| Visual Cognition | The user’s visual understanding of industrial structures. |
| Main Dimension | A structurally dominant dimension with strong industrial stability and cognition consistency. |
| Secondary Dimension | A structurally weaker dimension with lower long-term influence. |
| Product Cluster | A business-oriented grouping structure built upon industrial structural systems. |
| Industrial Structure | The stable structural logic underlying industrial products. |
| Structural Representation | The method by which industrial structures are expressed within systems or channels. |
| Industrial Cognition | The industrial understanding shared across engineering, manufacturing, supply chain, and customer systems. |
| Dimension Permutation | The theoretical combinations generated from multiple structural dimensions. |
| Structural Stability | The ability of a structure to remain valid over long industrial time cycles. |
| Industrial Structural World | The complete structural world formed by industrial dimensions, structures, and existence systems. |
| Channel Representation | The expression of industrial structures within specific websites or marketplaces. |
| MAP Structure | The organizational structure used by a specific channel to represent industrial structures. |
| Industrial Definition Authority | The authority to determine whether an industrial structure truly exists. |
| Structural Explosion | The uncontrolled expansion of unstable structural combinations or dimensions. |
